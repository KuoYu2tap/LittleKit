# LittleKit
一些小工具，使用Python写的，处理一些杂事
## CalculateForMOne
> 该项目是用于计算M1卡的校验和
### 应用场景
破解M1卡后，与金额相符的校验和HEX值
### 使用方式
输入金额数据，会生成相应的HEX值，该值替换M1卡内数据，写入后M1卡金额更新。

## SpiderPageCSV
